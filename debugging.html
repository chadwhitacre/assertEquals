3.4 Debugging Screen

The debugging screen is a primitive terminal for interacting with a Python debugger session. When a child process includes the string '(Pdb) ' in its output, assertEquals enters the debugging screen. When the debugger exits, assertEquals returns to the previous screen, ignoring any report output that may have followed the debugging session.

You can easily start debugging from any point in your program or tests by manually setting a breakpoint: 

import pdb; pdb.set_trace()
